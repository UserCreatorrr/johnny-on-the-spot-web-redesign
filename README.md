# Johnny on the Spot Website

Web corporativa experimental de Johnny on the Spot, construida con HTML, CSS, JavaScript, GSAP, ScrollTrigger y Lenis.

## Desarrollo local

Requiere Node.js.

```bash
node server.mjs
```

La web estará disponible en `http://localhost:4173`.

## Recursos audiovisuales

Los vídeos se almacenan con Git LFS para mantener todos los casos y el hero dentro del repositorio.
